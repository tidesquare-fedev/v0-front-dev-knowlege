# 구글 시트 API를 활용한 프로모션 페이지

이 프로젝트는 구글 시트의 마케팅 데이터를 실시간으로 가져와서 표시하는 프로모션 페이지 예제입니다.

## 🚀 주요 기능

- 구글 시트에서 실시간 데이터 로딩
- 투어비스 API를 통한 안전한 데이터 접근
- 반응형 디자인과 로딩 상태 표시
- 여러 섹션별 데이터 관리
- 서버 사이드 데이터 페칭

## 🛠️ 기술 스택

- **Next.js 14** - React 프레임워크
- **TypeScript** - 타입 안전성
- **Tailwind CSS** - 스타일링
- **Google Sheets API** - 데이터 소스
- **Server Actions** - 서버 사이드 로직

## 📋 구글 시트 구조

구글 시트는 다음과 같은 컬럼 구조를 가져야 합니다:

| A (상품명) | B (가격) | C (이미지 URL) | D (할인가) | E (원가) |
|------------|----------|----------------|------------|----------|
| 상품명     | 가격     | 이미지 URL     | 할인가     | 원가     |
| 제주도 패키지 | 299000 | https://... | 75000   | 374000 |

## 🔧 설정 방법

### 1. 환경 변수 설정

`.env.local` 파일에 API 키를 설정하세요:

```bash
API_KEY=your_jwt_token_here
```

### 2. 구글 시트 정보 설정

`src/app/page.tsx`에서 다음 정보를 실제 값으로 변경하세요:

```tsx
<PromotionSection
  title="🔥 인기 상품"
  documentId="your_actual_document_id"  // 실제 구글 시트 문서 ID
  sheetName="인기상품"                   // 실제 시트 이름
  cellStart="A"                          // 시작 셀
  cellEnd="E"                            // 끝 셀
/>
```

### 3. 구글 시트 시트 이름 설정

다음 시트들을 구글 시트에 생성하세요:
- `인기상품`
- `신규상품`
- `할인상품`
- `시즌상품`

## 🚀 실행 방법

```bash

pnpm install

# 개발 서버 실행
pnpm dev

# 빌드
pnpm build

# 프로덕션 실행
pnpm start
```

## 📁 프로젝트 구조

```
src/
├── app/
│   ├── actions.ts              # 서버 액션 (구글 시트 API 호출)
│   ├── components/
│   │   ├── PromotionCard.tsx   # 개별 상품 카드 컴포넌트
│   │   └── PromotionSection.tsx # 섹션별 상품 목록 컴포넌트
│   ├── layout.tsx              # 레이아웃
│   └── page.tsx                # 메인 페이지
```

## 🔒 보안 고려사항

- API 키는 서버 사이드에서만 사용됩니다
- 클라이언트에 민감한 정보가 노출되지 않습니다
- Server Actions를 통해 안전한 데이터 페칭을 구현했습니다

## 📱 반응형 디자인

- 모바일: 1열 그리드
- 태블릿: 2열 그리드
- 데스크톱: 3-4열 그리드
- 모든 화면 크기에서 최적화된 사용자 경험
