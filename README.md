# v0 프론트엔드 개발 지식 저장소

이 저장소는 **v0**에 입력할 프롬프트들을 모아놓은 개발 지식 공유 공간입니다.

## 📁 저장소 구조

### 📄 루트 마크다운 파일들

루트 디렉토리에 있는 `.md` 파일들은 **v0에 입력할 프롬프트 데이터**입니다. 각 파일은 특정 개발 주제나 기술에 대한 상세한 가이드와 예제 코드를 포함하고 있습니다.

현재 포함된 가이드:

- **[`google-sheet-api.md`](google-sheet-api.md)** - 구글 시트 API 사용 코드 작성 가이드
  - 투어비스 API를 통한 구글 시트 데이터 접근 방법
  - Next.js Server Action을 활용한 서버 사이드 API 호출
  - 프로모션/크로스셀 페이지 개발 예제

### 📁 examples 폴더

`examples` 디렉토리에는 **Next.js에서 실제로 사용하는 예제 프로젝트들**이 포함되어 있습니다.

#### 현재 예제:

- **[`promotion-page-using-google-sheet-api/`](./examples/promotion-page-using-google-sheet-api/)** - 구글 시트 API를 활용한 프로모션 페이지 예제

## 🚀 사용 방법

### 1. 프롬프트 데이터 활용

루트의 마크다운 파일들을 v0에 복사하여 붙여넣기하면, 해당 주제에 대한 상세한 개발 가이드를 받을 수 있습니다.

### 2. 예제 프로젝트 실행

각 예제 폴더의 README.md 파일 참고

## 💡 이 저장소의 특징

- **실용적**: 실제 프로덕션에서 사용 가능한 코드 예제 제공
- **체계적**: 주제별로 분류된 명확한 구조
- **즉시 사용 가능**: v0에 바로 입력하여 사용할 수 있는 프롬프트 형태
- **학습 친화적**: 이론과 실습을 모두 포함한 종합적인 가이드
